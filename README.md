Technology Stack

Frontend (Client-Side)
	•	Framework: React.js 
	•	Styling: Tailwind CSS / Bootstrap 
	•	State Management: Context API
	•	API Handling: Axios

Backend (Server-Side)
	•	Framework: Node.js with Express.js 
	•	Authentication: JWT (JSON Web Tokens)

Database (Storage)
	•	NoSQL: MongoDB 
	•	ORM/ODM: Mongoose (MongoDB)

Deployment & Hosting
	•	Frontend: Vercel 
	•	Backend: Render
	•	Database: MongoDB Atlas
